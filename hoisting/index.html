Hoisting Tutorial

hoisting is how javascript moves variables and functions to the top of their scope during compilation.(only decalration is hoisted)

    • var:- It is hoisted and initialialized as undefined.Befor initialization when trying to access it gives us undefined
      const,let and class:- It is hoisted but exists in temporal deadzone . whenever we are trying to access it before their initialization it gives us Refference Error
      function:- both decalration and defination is hoisted and we can call the function before declaration
      function Expression:- only varaible hoisted not the function
      Import Statements:- imports are hoisted initialization happend before the module code executes;
